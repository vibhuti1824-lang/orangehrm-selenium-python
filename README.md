# OrangeHRM User Management Automation (Selenium + Python)

## Overview
This project automates the end-to-end User Management workflow of the OrangeHRM demo application using Selenium with Python.

## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Page Object Model (POM)

## Application Under Test
- URL: https://opensource-demo.orangehrmlive.com
- Username: Admin
- Password: admin123

## How to Run
pip install -r requirements.txt
pytest

## Scenarios Covered
- Admin login
- Navigate to User Management
- Search user
